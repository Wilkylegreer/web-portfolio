Bulk File Renamer

This bulk file renamer was made for my CS 1030 class, since I'm a self taught programmer for around 8 years I pushed myself to create a UI to show the program off instead of just creating a simple text based program. This program does what is says, it renames up to thousands of files with a custom prefix that you can change.
